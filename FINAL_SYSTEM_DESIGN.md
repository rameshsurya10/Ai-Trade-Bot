## Key Differences from Auto-Trading

| Feature | Auto-Trading (NOT what you want) | Manual Trading (WHAT YOU WANT) |
|---------|----------------------------------|--------------------------------|
| **Who trades?** | Bot places orders automatically | YOU place orders manually |
| **Bot's role** | Executes trades | Gives signals & suggestions |
| **Your role** | Monitor only | Make final decision & execute |
| **Risk** | Bot can make mistakes | You control everything |
| **Exchange connection** | Bot needs API keys with trade permission | No trade API needed (read-only) |

## Configuration File

```yaml
# config.yaml - Your settings

# What to analyze
symbol: "BTC-USD"
timeframe: "1h"  # 1m, 5m, 15m, 1h, 4h, 1d

# Data source
data_source: "binance"  # binance, yahoo, etc.

# Signal thresholds
min_confidence: 0.55    # Only signal if confidence > 55%
min_risk_reward: 2.0    # Minimum 1:2 risk:reward

# Notifications
notifications:
  desktop: true         # Desktop popup
  sound: true           # Sound alert
  telegram: false       # Set true and add token for Telegram
  telegram_token: ""
  telegram_chat_id: ""

# Risk suggestions (for the signals you receive)
risk:
  stop_loss_atr_multiplier: 2.0    # Stop = 2 Ã— ATR
  take_profit_multiplier: 2.0      # TP = 2 Ã— stop distance

# Analysis
analysis:
  lookback_candles: 100            # How many candles for prediction
  features: 40                     # Number of indicators

# System
auto_reconnect: true               # Reconnect if data drops
max_reconnect_attempts: 3          # Then stop if can't reconnect
log_level: "INFO"                  # INFO, DEBUG, WARNING
```
